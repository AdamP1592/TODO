# Project Incubator
This is just a repository that stores a list of project descriptions I intend on tackling in the future.

## Projects:

### RAG Safety Lookup.
A retrieval system that aggregates data from global product safety databases to identify safety concerns for products. Creates effective search queries from user-defined requests, and generates confidence-aware safety reports based on source coverage and result relevance. Check out what I'll be doing [here](./projects/RAG_SAFETY_TOOL.md).


### RAG Resume Refiner
Instead of the standard "build it for them" a system that recommends specific content changes and provides reasoning for suggesting them, while also allowing for an underlying initial custom prompt to be written for providing the llm direction.

### Compute Cluster Digital Twin
A simulated compute cluster that allows me to operate as though each node is an independent part of a real network. Built for testing hpc software without a compute cluster.

### Neuron Growth Cone Dynamics Simulation
A simulation of the growth and movement of neurons over time built to integrate with Cable theory and Compartmental neuron models. 

Relates to my steady-state neuron culture simulation. [here](https://github.com/AdamP1592/Network-Simulation).

### Spatiotemporal Attention Transformer Model
Stablize the movement of a traditional motion governor(like MPCs) utilizing a transformer model trained to make adjustments to stabilize and smooth the motion. Done in simu, with rigid body dynamics.
